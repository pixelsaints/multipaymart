# Multipaymart Frontend

Frontend-only project using PHP (for includes) and Gulp (for scss and tailwind).

---

## Setup

1. Install dependencies
npm install

2. Run Gulp
gulp

3. Start MAMP and open
http://localhost:8888/multipaymart/

---

## Structure

- src/ → source files (SCSS, JS, images)
- assets/ → compiled files (do not edit)
- components/ → header, footer, etc.
- pages/ → page content
- index.php → router

---

## Notes

- PHP is used only for includes
- No backend / API / database
- Do not edit files inside assets/

---

## Build

gulp build
